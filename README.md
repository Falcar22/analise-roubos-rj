# Análise de Roubos em Áreas de UPP - RJ

## Objetivo
Realizar uma análise exploratória de dados públicos de segurança do Estado do Rio de Janeiro, identificando padrões temporais e territoriais de ocorrências de roubos em áreas com Unidades de Polícia Pacificadora (UPPs).

## Base de dados
Dados públicos disponibilizados pelo Instituto de Segurança Pública do RJ (ISP-RJ), contendo registros mensais por UPP.

## Etapas do projeto
- Tratamento e limpeza dos dados utilizando Python e Pandas
- Criação de série temporal (ano/mês)
- Análise da evolução histórica das ocorrências
- Identificação das UPPs com maior concentração de roubos
- Visualização gráfica com Matplotlib

## Principais conclusões
- Observou-se aumento significativo dos roubos entre 2015 e 2019
- Após 2019 ocorreu redução consistente nos registros
- A UPP Vila Kennedy acompanhou o padrão geral do estado, indicando fenômeno estrutural

## Tecnologias utilizadas
- Python
- Pandas
- Matplotlib
