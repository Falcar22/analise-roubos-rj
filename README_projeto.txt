Projeto: Análise de Evolução de Roubos em Áreas de UPP no RJ

Objetivo:
Analisar a evolução temporal dos registros de roubos em áreas com Unidades de Polícia Pacificadora (UPP) no estado do Rio de Janeiro.

Base de dados:
Dados públicos do Instituto de Segurança Pública (ISP-RJ), contendo registros mensais por UPP.

Etapas realizadas:
- Tratamento e limpeza dos dados
- Criação de série temporal (ano/mês)
- Análise de concentração territorial
- Identificação das UPPs com maior volume de ocorrências
- Análise temporal geral do estado
- Análise específica da UPP Vila Kennedy

Principais conclusões:
- Houve aumento significativo de roubos entre 2015 e 2019
- Após 2019 ocorreu redução consistente
- A UPP Vila Kennedy acompanha o padrão geral do estado, indicando fenômeno estrutural e não local isolado
“Projeto desenvolvido em Python (Pandas e Matplotlib) para prática de análise de dados e geração de visualizações a partir de dados públicos reais.”
